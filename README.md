## Компонент автоформы на Vue 3, который строит форму на основе переданной JSON-схемы.

## Установка и запуск проекта:

### Клонировать репозиторий:
```sh
git clone git@github.com:jull04/form.vue.git
```

### Установить зависимости:
```sh
npm install
```

### Запуск с Hot-Reload для Development среды

```sh
npm run dev
```

### Запуск для прода

```sh
npm run build
```
